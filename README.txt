There are two files in this folder. 


run_analysis.R:

The first one is run_analysis.R that contains the code for data processing. By running this file at the end you will have a tidy data table of averages of 561 features (columns) for each of the participants for each of the activities (rows).



Code_Book.Rmd:

The second file is Code_Book.Rmd that explains what are the input data and how it is processed using the code.
